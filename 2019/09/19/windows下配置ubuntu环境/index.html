安装 Oh My Zsh

安装主题 Powerlevel10k

  • 主题的路径在~/.oh-my-zsh/themes, 使某一主题生效的文件为~/.zshrc文件，找到ZSH_THEME="robbyrussell"一行（大概11行左右），把其注释掉，在下面添一行ZSH_THEME="Powerlevel10k"，之后关闭终端，再重启就好了。
  • 注意：在=符号右边一定不要有空格，否则会报错找不到此主题，我就是在这里困了半个多小时。 Then edit your ~/.zshrc and set ZSH_THEME="powerlevel10k/powerlevel10k".

Also add the following: POWERLEVEL9K_MODE="awesome-patched"
